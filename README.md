# My Kind of Dairy

A personal journal application that works entirely offline in the browser.

## Features

- Write and save journal entries with date selection
- Search through entries by date or content
- Export/import journal data as JSON
- Works completely offline with PWA support
- Dark theme with neon blue accents
- Responsive design for mobile and desktop

## Technologies Used

- HTML5
- CSS (Tailwind CSS)
- JavaScript
- localStorage for data persistence
- Service Worker for offline functionality

## How to Use

1. Open `mykindofdairy.html` in any modern web browser
2. Select a date using the date picker
3. Write your journal entry in the text area
4. Click the save icon to save your entry
5. Browse your entries below or search for specific content
6. Use the export/import buttons to backup or restore your journal

## Development

This is a client-side only application with no external dependencies except for Tailwind CSS which is loaded from CDN.
